# 🎨 Personal Portfolio – Trí

## 📌 Introduction
This is a personal portfolio website built to showcase projects, skills, and creative direction in architecture and design.  
The interface follows a modern, minimal style with a strong focus on visual hierarchy and layout clarity.

---

## 🚀 Objectives
- Present design thinking and architectural ability  
- Showcase personal projects in a visual way  
- Build a professional personal brand  
- Serve as a foundation for future portfolio development  

---

## 🧩 Structure

### 1. Home
- Split layout design (left–right)
- Large typography background (PORTFOLIO)
- Hero section with personal identity

### 2. About
- Personal introduction  
- Career orientation  
- Skills list:
  - Revit  
  - Photoshop  
  - Illustrator  
  - 3Ds Max  

### 3. Contact
- Basic contact information  
- Email / Phone number
